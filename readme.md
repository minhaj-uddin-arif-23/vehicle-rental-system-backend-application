Vehicle Rental System & https://vehicle-rental-system-ochre.vercel.app/

## Features & Technology Stack

---

### Features

User authentication using JWT with role-based access (Admin & Customer)

Secure password hashing using bcrypt

Vehicle inventory management with availability tracking

Customer account and profile management

Booking system with:

Vehicle availability validation

Automatic rental cost calculation

Booking cancellation and return handling

Role-based data access (Admin: all data, Customer: own data only)

Modular and scalable backend architecture

Proper error handling and validation

---

### Technology Stack

Node.js – Server-side runtime

TypeScript – Type-safe development

Express.js – REST API framework

PostgreSQL – Relational database

bcrypt – Password hashing

jsonwebtoken (JWT) – Authentication & authorization
